# Footer

Page footer section with border separator.

## Basic usage

```blade
<tabler:footer>
    <span>&copy; 2024 My App</span>
    <nav class="flex gap-4">
        <a href="/privacy">Privacy</a>
        <a href="/terms">Terms</a>
    </nav>
</tabler:footer>
```

## Props

No props. Uses the default slot for content. Renders a `<footer>` with flex layout and top border.
